# 视频教程

> 配合 [使用手册.md](使用手册.md) 食用，本页只列视频录制要点和命令速查。

---

## 环境准备（5分钟）

```bash
# 1. 克隆仓库
git clone https://github.com/jiangkaili/binance-trader-bot.git
cd binance-trader-bot

# 2. 创建虚拟环境（Python 3.10+）
python3 -m venv .venv
source .venv/bin/activate    # Linux/macOS
# .venv\Scripts\activate     # Windows PowerShell

# 3. 安装依赖
pip install -r requirements.txt
```

---

## 配置（5分钟）

1. 在币安官网创建 API Key（仅开合约交易权限）
2. 复制 `.env.testnet` 为 `.env`，填入密钥：

```bash
BINANCE_API_KEY=你的key
BINANCE_API_SECRET=你的secret
USE_TESTNET=true              # 先用测试网
```

3. 策略参数在 `config/trader.yaml`，默认值已调好，不用改。

---

## 运行（3分钟）

```bash
# 测试网 dry-run（不下真单，只看信号）
python scripts/live_trader.py --dry-run

# 测试网实盘（测试资金，不花真钱）
python scripts/live_trader.py --env-file .env.testnet

# 主网实盘（真金白银，请先读完使用手册）
python scripts/live_trader.py --env-file .env.prod
```

---

## 监控与停止

```bash
# 查看实时日志
tail -f data/live_trader.log

# 查看持仓和算法订单
python scripts/positions_futures.py
python scripts/list_algo_orders.py

# 优雅停止（自动平仓后退出）
kill -TERM <pid>

# 紧急熔断（停止开新仓，不平已有仓）
touch data/KILLSWITCH
```
